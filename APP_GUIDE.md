# Kioku Guide: How to Use Your AI Memory Assistant

Welcome to **Kioku**, your high-productivity digital brain. Kioku is designed to help you organize thoughts, remember personal facts, and manage tasks through natural conversation.

## üöÄ Core Features

### 1. Unified Chat Interface
- **Natural Conversation**: Talk to Kioku just like a person.
- **Voice Input**: Tap the microphone icon for hands-free interaction.
- **Contextual Awareness**: Kioku remembers your past messages and preferences.

### 2. Intelligent Memory (Digital Brain)
- **Automatic Learning**: Kioku automatically extracts facts from your chats (e.g., "I like dark chocolate").
- **Explicit Memory**: Tell Kioku "Remember that I use VS Code" to force a memory entry.
- **Privacy**: All memories are stored locally on your device in IndexedDB.

### 3. Smart Notes & Knowledge Base
- **Markdown Support**: Take rich notes with full markdown formatting.
- **Vector Search**: Kioku uses local vector embeddings to find relevant notes when you ask questions.
- **Organization**: Pin important notes and search through your entire knowledge base.

### 4. Natural Language Reminders
- **Set it and Forget it**: Just say "Set a reminder for my meeting at 3 PM tomorrow".
- **Dynamic Scheduling**: Kioku understands relative time (e.g., "in 2 hours").
- **Tags**: Use `[REMINDER: "task" AT "time"]` if you want to be precise, but Kioku usually handles it for you.

### 5. Web Search & Real-time Info
- **Live Knowledge**: If enabled in Settings, Kioku can search the web to answer current event questions.
- **Source Citation**: Web results are cited with original URLs.
